SINHALEARN

Currently working on publishing an Android app, SinhaLearn, through the Google Play Store (Currently In-Review).
I was frustrated with the lack of good resources to learn the Sinhalese language online and thought to build my own mobile app that incorporates my favourite language learning method, cue cards.
Built using React-Native using a Firebase Realtime Database to store data for cue card style lessons.
